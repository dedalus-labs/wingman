# Wingman

AI coding assistant for the terminal.

## Quick Start

```bash
# Clone
git clone https://github.com/dedalus-labs/wingman.git
cd wingman

# Install dependencies
pip install textual dedalus-labs

# Run
python -m wingman
```

Get your API key at [dedaluslabs.ai/dashboard/api-keys](https://dedaluslabs.ai/dashboard/api-keys)
