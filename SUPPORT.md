### Questions

**Before opening an issue:**

1. Search [existing issues](https://github.com/dedalus-labs/wingman/issues)
2. Check if you're on the latest version

**Good places to ask:**

- [GitHub Issues](https://github.com/dedalus-labs/wingman/issues) — Bugs, feature requests

### Bugs

Found a bug? [Open an issue](https://github.com/dedalus-labs/wingman/issues/new).

Include:

- Python version (`python --version`)
- Wingman version (`wingman --version` or `uv run wingman --version`)
- Terminal emulator (iTerm2, Ghostty, Terminal.app, etc.)
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs/tracebacks

### Security Issues

**Do NOT open public issues for security vulnerabilities.**

Email [security@dedaluslabs.ai](mailto:security@dedaluslabs.ai). See [SECURITY.md](SECURITY.md).
